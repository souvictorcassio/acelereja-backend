# AcelerEJA

Um canal que auxilia jovens e adultos a retomarem os estudos, facilitando o encontro com 
instituições de ensino próximas e acessíveis, promovendo inclusão e valorizando suas 
trajetórias.

# Sobre o projeto

## Contexto

Muitos jovens e adultos, após períodos afastados da escola, buscam retomar seus estudos, 
motivados pela necessidade de concluir o Ensino Fundamental e Médio, etapas essenciais 
para a inserção no mercado de trabalho, a conquista de melhores oportunidades 
profissionais e a ascensão social.

Dessa forma, o projeto busca não apenas criar uma solução tecnológica, mas também 
contribuir para a valorização da Educação de Jovens e Adultos como instrumento de 
inclusão, emancipação e transformação social, reconhecendo as instituições de ensino 
como protagonistas desse processo e fortalecendo o compromisso com uma educação pública mais acessível, acolhedora e eficiente.

- JavaScript
- Prisma
- SQLite (dev)
- PostgreSQL (produção)

Instalação

1. Clone o repositório
   ```bash
   git clone https://github.com/souvictorcassio/acelereja-backend.git
2. Entre na pasta do projeto
   ```bash
   cd backend
3. Instale as dependências
   ```bash
   npm install
4. Entre na sub pasta
   ```bash
   cd src
5. Rode o servidor local
   ```bash
   node server.js

*Certifique-se de possuir uma base de dados PostgreSQL para utilização do código.
